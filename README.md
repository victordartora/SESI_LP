Comandos básicos do Git
# Configuração
git config --list

Mostra todas as configurações do Git no seu ambiente (nome, email, editor, etc).

git config --global user.name "Seu Nome"

Define o seu nome para os commits.

git config --global user.email "seu@email.com"

Define o seu email para os commits.

# Clonar repositório
git clone <url>

Copia um repositório remoto (ex: do GitHub) para o seu computador.

# Ver status
git status

Mostra o estado atual do repositório:

Arquivos modificados
Arquivos novos
O que já está pronto para commit

# Adicionar arquivos
git add <nome-do-arquivo>

Adiciona um arquivo específico para o próximo commit.

# Salvar mudanças
git commit -m "descrição"

Salva as alterações com uma mensagem explicando o que foi feito.

# Enviar para o repositório remoto
git push

Envia ps commits para o servidor remoto (ex: GitHub).

# Atualizar repositório local
git pull

Baixa as mudanças do repositório remoto e atualiza o seu projeto local.
